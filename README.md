# calculadora-input-html

## Descripción
Este programa consiste en una calculadora básica que permite realizar operaciones matemáticas simples mediante una interfaz gráfica interactiva. Se utiliza HTML, CSS y JavaScript para simular el funcionamiento de una calculadora digital real.

## Funcionamiento
- Es una aplicación web.
- Funciona en el navegador.
- Permite ingresar números mediante botones (no por teclado).
- Permite realizar las siguientes operaciones:
  - suma (+)
  - resta (-)
  - multiplicación (*)
  - división (/)
- Incluye botón Reset para limpiar toda la operación.
- Incluye botón Del para borrar el último dígito ingresado.
- Solo permite números enteros (no decimales).

## ¿Cómo ejecutar el programa?
- Clonar el repositorio en el escritorio.
- Abrir el archivo index.html en el navegador.
- Utilizar los botones de la calculadora para ingresar los valores.
- Presionar "=" para obtener el resultado.
